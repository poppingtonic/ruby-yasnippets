## Ruby YASnippets 

This is a collection of custom snippets for use with YASnippets. Save
this folder in 

``` ~/.emacs.d/custom/ ``` 

and add the following to
your ```.emacs``` or ```init.el```:

```lisp

    (load "~/.emacs.d/custom/ruby-yasnippets/ruby-yasnippets.el")
    (require 'ruby-yasnippets)

```
